Sistema Escolar Fullstack

Este proyecto es una aplicación web completa para la gestión escolar, diseñada para facilitar la administración de usuarios y eventos académicos.

Tecnologías Utilizadas

Frontend: Desarrollado con Angular (versión 16), utilizando TypeScript, HTML y SCSS para crear una interfaz moderna y responsiva. Se integra Angular Material para componentes visuales elegantes y Bootstrap para el diseño de la cuadrícula.

Backend: Construido con Django (Python) y Django REST Framework para crear una API robusta y segura.

Base de Datos: Compatible con MySQL (para desarrollo local con XAMPP) y PostgreSQL (para producción en Render).

Características Principales

Gestión de Usuarios: Registro, edición y eliminación de Administradores, Maestros y Alumnos con roles y permisos diferenciados.

Autenticación Segura: Sistema de Login con tokens para proteger el acceso.

Eventos Académicos: Módulo completo para registrar y visualizar eventos, con validaciones de fechas y horarios.

Gráficas Estadísticas: Visualización de datos clave mediante gráficas interactivas (barras, pastel, líneas).

Diseño Adaptativo: Interfaz amigable que se ajusta tanto a computadoras de escritorio como a dispositivos móviles.

Despliegue

El sistema está configurado para desplegarse fácilmente en la nube:

Frontend: Alojado en Vercel.

Backend y Base de Datos: Alojados en Render.
